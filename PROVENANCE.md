# Repository provenance

This standalone repository was split from the historical
sergkurilenko/research repository. The canonical source snapshot was commit
7695359803e9d037c3ff34d07ec2ef45cea9f1ba on branch
agent/journal-revision-artifact.

The public main branch intentionally starts with a clean root commit so that
the Git history does not contain SHARD/article5 code or experimental
artifacts. The historical repository remains archived and read-only as a
migration index, preserving the original branches and commit identifiers.
